# Índice de Condiciones Habilitantes para la IA — Infraestructura Digital

Dashboard interactivo que muestra la dimensión de infraestructura digital del Índice de Condiciones Habilitantes para la IA por provincia argentina.

## Indicadores

- **Penetración de internet fijo** (50%): Accesos cada 100 hogares
- **Velocidad media de descarga** (30%): Mbps promedio
- **Proporción de fibra óptica** (20%): % de accesos por fibra

## Metodología

Cada indicador fue normalizado con el método Min-Max (escala 0-100) y luego ponderado según su relevancia.

## Fuente de datos

ENACOM (Ente Nacional de Comunicaciones)

## Deploy

Sitio estático deployado en Vercel.
