# Indice Comparativo de Infraestructura Digital Provincial

Este índice forma parte del **Indice Comparativo de Condiciones Habilitantes para la IA**, un proyecto en construcción que busca medir las condiciones estructurales de cada provincia argentina para adoptar y aprovechar herramientas de inteligencia artificial.

El índice general tiene tres dimensiones: **infraestructura digital**, **capital humano** y **tejido productivo**. Este dashboard presenta la primera de ellas.

## Qué mide

Compara las condiciones de conectividad de cada provincia a partir de tres indicadores:

- **Penetración de internet fijo** (50%): Accesos cada 100 hogares por provincia
- **Velocidad media de descarga** (30%): Mbps promedio por provincia
- **Proporción de fibra óptica** (20%): Proporción de accesos por fibra sobre el total de accesos

## Metodología

Cada indicador fue normalizado con el método Min-Max (escala 0-100) para hacerlos comparables, y luego se ponderaron según su relevancia. El índice final es la suma ponderada de los tres componentes normalizados.

Al ser un índice relativo, no indica si una provincia "está habilitada" en términos absolutos, sino cuán habilitada está en comparación con el resto.

## Fuente de datos

ENACOM — Ente Nacional de Comunicaciones (3T 2025). Elaboración propia.
